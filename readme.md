`战绩查询`<br><br>
[`.盒`+`ID`] PC战绩查询<br><br>
[`.2042战绩`+`ID`] PC战绩查询<br><br>
[`.2042xbox端战绩`+`ID`] xbox战绩查询<br><br>
[`.2042ps端战绩`+`ID`] ps战绩查询

**需要安装pillow**

    pip install pillow
